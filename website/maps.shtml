The Maps

Mappa Mercia is not just another free map. It is (like OpenStreetMap) about free geo data that you can use in whatever way you can imagine. Maps are just one thing you can create with it.

This page shows a number of different maps for Birmingham which are available on Mappa Mercia. To find out about other ways to use the geo data of Birmingham have a look at the "use the map" page.

  • Standard Map

    An all purpose map of Birmingham. Apart from the streets it shows the canals and many points of interest.

  • Cycle Map

    This map is made for cyclists. It has contour lines, shows the routes of the National Cycle Network and lots of other information which is important for cyclists!

  • Public Transport Map

    A map for public transport users in the West Midlands. It displays public transport information like bus stops and bus routes which are stored in the OpenStreetMap database. This map is kindly provided by öpnvkarte.de.

  • Blue Plaques Map

    Birmingham has many blue plaques to commemorate famous inhabitants. This map shows the blue plaques which have been mapped already.
