The Maps

Mappa Mercia and OpenStreetMap are about free geo data that you can use in whatever way you want.

  • Standard Map

    An all purpose map of Birmingham.

  • Cycle Map

    This map is made for cyclists. It has contour lines, shows the routes of the National Cycle Network and lots of other information which is important for cyclists.

  • Public Transport Map

    This map is provided by öopvkarte.de. It displays public transport information like bus stops and bus routes which are stored in the OpenStreetMap database.
