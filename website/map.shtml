Welcome!

This is Mappa Mercia: a project to digitally remap the West Midlands in a free and open format.

The map you see here was created by volunteers and it is free for everyone to use and to edit. Mappa Mercia is part of OpenStreetMap. OpenStreetMap is dedicated to allow you to view, edit and use geographical data in a collaborative way from anywhere on Earth.
