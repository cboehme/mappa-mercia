Gritting Map

This map shows which roads in Birmingham are gritted when it is cold and frosty.

Currently, OpenStreetMap has only gritting route data for Birmingham. Get in touch with the Birmingham mappers to find out how to add the routes in your town to the map.

There are not many grit bins on the map yet. If you know of any missing ones report an error or, even better, add the bins yourself.
